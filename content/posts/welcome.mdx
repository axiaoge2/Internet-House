# 欢迎来到我的小屋

这是我使用Next.js 16、Tailwind CSS和MDX构建的个人小屋网站。

## 技术栈

- **Next.js 16**: React框架，提供服务端渲染和静态生成
- **Tailwind CSS v4**: 现代化的CSS框架
- **MDX**: 支持在Markdown中使用React组件
- **TypeScript**: 类型安全的JavaScript

## 功能特性

1. 文章列表展示
2. 文章详情页
3. 分类和标签系统
4. 全文搜索
5. 代码高亮
6. 响应式设计
7. 暗黑模式

## 代码示例

这里是一个简单的TypeScript代码示例：

```typescript
interface User {
  name: string;
  email: string;
}

function greetUser(user: User): string {
  return `Hello, ${user.name}!`;
}

const user: User = {
  name: "张三",
  email: "zhangsan@example.com"
};

console.log(greetUser(user));
```
